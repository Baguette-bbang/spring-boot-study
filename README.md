### **앞으로 이어질스프링 완전 정복 시리즈**

스프링을 완전히 마스터할 수 있는 시리즈가 이미 준비되어있다. 실제 실무에서 사용하는 핵심 스프링 기능을 중심으로, <br>
실무에서 쓰이지 않거나 오래된 기능은 과감하게 생략한 실무 노하우를 전수해드릴 예정이라고 한다. <br>
물론 강의를 찍은지도 꽤 시간이 지나서 트렌드가 약간은 달라졌을테지만… ㅎ
